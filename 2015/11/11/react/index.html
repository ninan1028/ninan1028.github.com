react 入门

来源 阮一峰react入门教程

1 React 安装

react的安装包,可以在官网下载.阮一峰老师所写的文档中有一个代码demos,在demos中包含了react源码.安装方式
$ git clone git@github.com:ruanyf/react-demos.git

如果没有安装git,可以直接下载,zip压缩包

2 HTML 模板

使用React的网页源码,结构大致如下

<!Doctype html>
<html>
<head>
    <title></title>
    <script type="text/javascript"  src="../bulid/react.js"></script>
    <script type="text/javascript"  src="../bulid/react-dom.js"></script>
    <script type="text/javascript"  src="../bulid/boowser.min.js"></script>
</head>
<body>
<div id="example"></div>
</body>
<script type="text/babel">
   //code
</script>
</html>

需要注意的地方 ,最后一个script标签的type属性为 text/babel. 这是React独有的JSX语法,跟javascript不兼容.凡是使用JSX的地方,都要加上type=text/babel””
